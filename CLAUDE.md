This is a chrome extension that works on the acm dl website. 

The problem is that if you're not signed in to your account, acm dl opens pdfs of papers in their native viewer, which I don't like. I'd like it to instead open in the default browser pdf viewer (google scholar reader for me). The trick is to simply replace "epdf" with "pdf" in the paper url. 

For example, here is a paper pdf for EgoTouch: https://dl.acm.org/doi/10.1145/3654777.3676455. 
If I click the default  PDF/eReader button (in red), it opens the following link: https://dl.acm.org/doi/epdf/10.1145/3654777.3676455. Instead, make it point to https://dl.acm.org/doi/pdf/10.1145/3654777.3676455. That is all.
