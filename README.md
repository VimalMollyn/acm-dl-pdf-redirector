# ACM DL PDF Redirector

When you're not signed in, the ACM Digital Library opens papers in its own eReader. This swaps the `epdf` in the link for `pdf` so they open in your browser's PDF viewer instead.

Install from the [Chrome Web Store](https://chromewebstore.google.com/detail/jceolpmanpoihdonffihimomdldglonp).

## Running from source

Open `chrome://extensions`, turn on Developer mode, and click "Load unpacked" on this folder.
